# Sparkz v1 scope

The full three-in-one architecture (Creator Coin + Culture Coin + a 15-module Meme Engine with Alpha Radar) is the north star. This file is what actually ships first, and what is deliberately deferred. The rule: ship a thin slice, prove one loop, let the culture pull the rest.

## v1 = one path, one loop

**First launch: Zoostr, as a Creator Coin. Culture Coin path is deferred.**

### In v1

- **Creator Coin path only.** One front door. The Culture Coin path (stewards, community constitution, contributor governance, treasury) is deferred - it needs governance + the MIDAO legal wrapper first.
- **The spark (pre-token).** Project profile + a Creator Capsule + collectables + the boost/leaderboard + backing. No token required to start.
- **Fiat-or-BYOK onboarding.** Back with a card, no wallet (the #1 priority), or bring your own AI key. Non-technical creators draw compute from the Sparkz treasury (the treasury-funds-onboarding loop).
- **Creator-first economics, if/when a token launches.** 1/1/98 default (creator up to 98%, 1% agent-upkeep floor, 1% community-treasury floor), deployed via a Clanker adapter with a 0xSplits distribution wallet so the creator's share routes to collaborators.
- **Meme Engine = ONE thin loop, human-in-the-loop only (Manual/Assisted tier).** A human or ZOL flags a moment -> the engine drafts ~3 native responses grounded in the Creator Capsule -> human approves -> post -> **Community Swarm** (holders remix approved templates in a time-boxed mission) -> reward the best + write a receipt. NO Alpha Radar, NO autonomous posting.
- **Receipts** (DreamNet) on contributions.

### The v1 proof

Does the thin Community-Swarm loop turn Zoostr's backers into an active cultural distribution network instead of passive holders? If yes, the whole thesis is validated and you have earned the right to build Alpha Radar. If no, no amount of trend-detection ML saves it.

## Deferred (north star, NOT v1)

- **Culture Coin launcher** (governance, treasury, contributor governance, stewards) - after the MIDAO/Greg legal wrapper.
- **Alpha Radar** - real-time multi-platform velocity/trend detection + the Trend Graph + the Alpha Desk. The big data system. This is the eventual moat but it is not what proves the thesis.
- **Autonomous posting tier** - v1 is human-in-the-loop; auto-posting to trends is high-embarrassment and waits until the Culture-Fit gate is trustworthy.
- **The rest of the 15 Meme Engine modules** (Mutation Lab, Opportunity Scout, Federation, narrative-mode campaigns, cross-platform native variants at scale).
- **Boosters-can-be-other-sparkz** - the composable creator-to-creator backing graph; build it in v1.1 once the single-creator loop works.

## Why this sequencing

The Meme Engine is ~80% of the build and 100% of the differentiation, so it gets prototyped FIRST - but as the smallest end-to-end loop, not the full trend-detection system. Culture Coin adds regulatory surface, so the simpler Creator Coin ships first. The architecture's own instinct - "keep Zaal's work, modularize it" - is honored by sequencing, not by building all three in parallel.
