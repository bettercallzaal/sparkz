# Nounish Prof x Zaal - Sparkz brainstorm notes

**2026-07-22 - Zabal Games brainstorm on GM Farcaster, ~39 min.** Speakers correctly
attributed from diarization (S1 = Zaal, S2 = Nounish Prof). This page is the Sparkz
takeaways only - what to build, improve, and remember.

## What to build / improve in Sparkz

1. **Transparency about fees, front and center.** When a creator graduates a no-token
   empire into an empire token, ~20% goes to the Empire Builder team. Show that plainly
   plus "what do you get for it, is it worth it?" - the creator decides. No vendor lock.
   (Already in the README pitch; make it visible in the product/graduation flow.)

2. **The on-ramp path is the product.** The concrete ladder Zaal wants to lead creators up:
   - Make an empire (tokenless).
   - Add a basic leaderboard.
   - With ~$25, make a channel + a leaderboard around who talks in that channel
     (Empire Builder can spin up a channel-specific empire).
   - Devs: spin up a small game or a custom leaderboard, pull in your own data.
   - Mirror any custom leaderboard into Empire Builder and distribute rewards off it.
   Sparkz is the integration glue between these, not a replacement. This is a real
   product roadmap - the stages of a spark before a token is even a question.

3. **Agent options, creator's choice.** Surface multiple agent backends in Sparkz, do
   not lock to one: **Clanker droids, ElizaOS, OpenClaw, Hermes.** Let a brand run one
   of each and compare.
   - Droids caveat: today you do NOT own the FID, and there is no BYOK (bring your own
     key) for token/API credits. Zaal wants BYOK because he is not taking profits.
   - Watch where droids are going (ask Quazia), not where they are now.

4. **Community "soul code" agent (Zaal's idea).** The top token-holder / top
   contributor of the day can add a line of "soul code" to the agent's personality -
   "today, be more like this." The agent evolves daily with the community's flavor.
   This is a real, non-speculative reason to hold the token: it is a tool you shape,
   not a bet.

5. **The "cheeky intern" agent voice (Nounish Prof's steer).** A brand agent should
   REPLY to others on the timeline - not spam top-level "look at me" casts. A little
   cheeky, a little funny, but genuinely interacting. Be more human. Keep
   human-in-the-loop: the agent drafts a cast, the human edits or kills it before it
   goes out (how droids work, and how the Sparkz agent already defaults). Reference:
   their agent "Warpy" has a practical personality (ask it "when did we talk about X on
   GM Farcaster" and it jumps to the timestamp).

6. **Tip for PRs, like Farcaster tips for casts.** Sparkz already leans open-source; the
   idea is to reward code contributions (PRs) the way Farcaster rewards content. Ties to
   the open-source repo + the "back the work" thesis.

7. **Educational on-ramp for non-devs.** Teach non-technical creators to put a first repo
   on GitHub and try vibe coding on a problem they actually have. (Aligns with the
   non-dev README + Glossary work already shipped.)

## Principles reinforced

- **Token was pushed on people** - Base's content-coin meta only ever offered "launch a
  token, we'll help you." Sparkz exists because that is the wrong first move.
- **DWR's test: "what would the token do?"** - Nounish Prof's frame: Sparkz IS the
  process of answering that before you launch. Until there is an answer, no token.
- **No vendor lock = the moat.** The Zora example: they felt pushed to launch a token,
  chose not to, and a year and a half later that was right. Keep creators un-locked -
  decentralize the audience, keep full ownership, bring profit/data/IP back to the artist.
- **Charlie Munger (via Prof): "show me the incentive, I'll show you the outcome."**
  Design the incentive; that is the whole retention game no one has solved.
- **The origin:** this all started from a musician asking Zaal for album funding - "I have
  a token idea but I do not want to just do it." A Capsule is the culmination of the idea
  (token or tokenless, a repo, content, a music drop).
